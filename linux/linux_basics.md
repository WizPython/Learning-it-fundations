# Linux Basics

## What is Linux?
Linux is an open-source operating system widely used in servers, cloud environments, and cybersecurity.

## Why Linux is important?
- Most cloud servers run Linux
- Cybersecurity tools are Linux-based
- Automation and scripting rely heavily on Linux

## Basic Commands
- pwd – show current directory
- ls – list files
- cd – change directory
- mkdir – create directories
- touch – create files
- rm – remove files
- cp – copy files
- mv – move or rename files

## Permissions (intro)
- r (read)
- w (write)
- x (execute)
